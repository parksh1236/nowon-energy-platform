# 노원구 에너지정보플랫폼

건축물 총괄표제부 × 월별 전기·가스 사용량 데이터를 지번주소로 매칭해
카카오지도 위에 시각화하는 웹 플랫폼입니다.

- **웹앱**: `platform/index.html` (정적 파일, 서버 불필요)
- **데이터 빌드**: `python3 platform/build_data.py` → `platform/data.json` 재생성
- **좌표 캐시**: `platform/coords.json` (카카오 지오코딩 결과, 3,377개 주소)
- **원본 데이터**: 국토부 건축물대장 총괄표제부, 전기·가스 에너지 사용량 (2025.12 ~ 2026.03)

## 사용법
1. 카카오 개발자 콘솔에서 JavaScript 키의 웹사이트 도메인에 배포 도메인 등록
2. `platform/` 폴더를 정적 호스팅(GitHub Pages 등)에 배포
